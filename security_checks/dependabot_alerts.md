# Dependabot Alerts
Ensures repository Dependabot security alerts are enabled

```yaml
dependabot_alerts:
  behavior: review
```

## Configuration

_None_

## Description
Dependabot alerts ensure that potential vulnerabilities in dependencies are being surfaced quickly for remediation.
